## Getting Started Checklist

### Week 1-2: Learn
- [ ] Read: "The Book on Rental Property Investing"
- [ ] Watch: BiggerPockets YouTube basics
- [ ] Join: BiggerPockets forums
- [ ] Calculate: Run numbers on our properties

### Week 3-4: Prepare
- [ ] Get pre-approved for loan
- [ ] Set up LLC (optional)
- [ ] Open separate business account
- [ ] Research property managers

### Month 2: Execute
- [ ] Tour properties
- [ ] Submit offers
- [ ] Get inspections
- [ ] Close

### Month 3+: Manage
- [ ] Find tenants
- [ ] Set up leases
- [ ] Set up accounting
- [ ] Scale: Repeat
